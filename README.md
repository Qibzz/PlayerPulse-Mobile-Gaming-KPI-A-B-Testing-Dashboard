PlayerPulse Analytics is a full-stack data analytics project using the Cookie Cats dataset (~90,000 players) from Kaggle. The project covers the entire analytics pipeline — from raw data ingestion to interactive multi-platform dashboards — with a focus on KPIs relevant to mobile gaming: retention, churn, monetization, and A/B testing.
